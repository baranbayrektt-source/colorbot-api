🚀 KULLANIM ADIMLARI:

1️⃣ GITHUB'A YÜKLEME:
   - push_to_github.bat dosyasına çift tıklayın
   - Otomatik olarak GitHub'a push eder
   - GitHub Actions tetiklenir

2️⃣ BUILD İZLEME:
   - GitHub'da Actions sekmesine gidin
   - https://github.com/baranbayrektt-source/colorbot-api/actions
   - Build durumunu takip edin

3️⃣ DOWNLOAD:
   - Build tamamlandığında
   - Actions > Latest workflow run
   - Artifacts > QUARXV1-Setup
   - Download ZIP

🎯 MANUEL TETIKLEME:

GitHub'da manuel olarak build tetiklemek için:
1. GitHub repository'ye gidin
2. Actions sekmesine tıklayın
3. "Build QUARXV1" workflow'unu seçin
4. "Run workflow" butonuna tıklayın
5. Branch seçin (main)
6. "Run workflow" ile başlatın
