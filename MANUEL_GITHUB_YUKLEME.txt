🚀 GITHUB ACTIONS KURULUMU

Git kurulu olmadığı için manuel olarak GitHub'a yükleyeceğiz.

📋 ADIM ADIM YÜKLEME:

1️⃣ GITHUB REPOSITORY:
   - https://github.com/baranbayrektt-source/colorbot-api
   - Repository'ye gidin

2️⃣ DOSYA YÜKLEME:
   - "Add file" > "Upload files" tıklayın
   - Tüm ColorBot klasörünü sürükleyin
   - Commit message: "QUARXV1 GitHub Actions Build System"
   - "Commit changes" tıklayın

3️⃣ GITHUB ACTIONS TETIKLEME:
   - Actions sekmesine gidin
   - "Build QUARXV1" workflow'unu seçin
   - "Run workflow" butonuna tıklayın
   - Branch: main
   - "Run workflow" ile başlatın

4️⃣ BUILD İZLEME:
   - Build durumunu takip edin
   - Süre: 5-10 dakika
   - Yeşil tik = başarılı

5️⃣ DOWNLOAD:
   - Build tamamlandığında
   - "Artifacts" > "QUARXV1-Setup"
   - "Download" ile ZIP indirin
